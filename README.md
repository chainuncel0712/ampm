# Comprehensive Multi-Agent AI Framework

This repository contains a multi-agent AI framework system with various components for intelligent routing and sub-agent management.

## Features
1. **Main Agent**: Intelligent decision-making and message routing.
2. **Sub-Agent Management**: Manage multiple sub-agents effectively.
3. **Telegram Integration**: Interact through predefined commands.
4. **REST API**: For agent communication.
5. **Conversation History**: Keep track of interactions.
6. **Multiple Agent Types**: General, coding, analysis, translation, and writing.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/chainuncel0712/ampm.git
   cd ampm
   ```

2. Set up your environment:
   ```bash
   cp .env.example .env
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:
   ```bash
   python app.py
   ```

## Commands
- `/list`: List all agents.
- `/create`: Create a new agent.
- `/chat`: Start a chat with an agent.
